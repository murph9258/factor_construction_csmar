# factor_construction_csmar
use csmar dataset, constructing asset pricing factors

【数据来源】CSMAR

单表查询->股票市场系列->股票市场交易->个股交易数据（日/周/月）/股票复权交易数据（日/周/月）

单表查询->股票市场系列->股票市场衍生指标->风险因子

单表查询->因子研究系列->fama french 因子（五因子）

单表查询->因子研究系列->已实现指标（相对报价价差，成交金额加权平均）

单表查询->公司研究系列->财务报表->资产负债表/利润表/现金流量表

单表查询->公司研究系列->财务报表附注->研发费用/营业收入（毛利）

单表查询->公司研究系列->财务指标分析->盈利能力/现金流分析（折旧摊销）

单表查询->公司研究系列->财务指标分析->披露财务指标

单表查询 /公司研究系列 /上市公司研发创新->研发投入情况表

【数据表构造】
1. 下载上市公司基本信息
2. 对每个上市公司，匹配2000-2025年的四个季度
